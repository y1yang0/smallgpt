# SmallGPT

A minimal GPT implementation from scratch for learning purposes.

## Model Architecture

```
                    ┌─────────────────────┐
                    │    Input Tokens      │
                    │     (seq_len,)       │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │  Token Embedding     │
                    │   (50257, 1024)      │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │ Position Embedding   │
                    │    (1024, 1024)      │
                    └──────────┬──────────┘
                               │
                        tok_emb + pos_emb
                               │
          ┌────────────────────▼────────────────────┐
          │          Transformer Block × 2          │
          │                                         │
          │    ┌─────────────────────────────┐      │
          │    │  LayerNorm 1    (1024,)     │      │
          │    └──────────────┬──────────────┘      │
          │                   │                     │
          │    ┌──────────────▼──────────────┐      │
          │    │  Attention                  │      │
     ┌────┤    │  Q,K,V: (1024, 1024)       │      │
     │    │    └──────────────┬──────────────┘      │
     │    │                   │                     │
  residual│              ⊕ ◄──┘ + x                 │
     │    │                   │                     │
     └────┤    ┌──────────────▼──────────────┐      │
          │    │  LayerNorm 2    (1024,)     │      │
          │    └──────────────┬──────────────┘      │
          │                   │                     │
          │    ┌──────────────▼──────────────┐      │
     ┌────┤    │  FeedForward                │      │
     │    │    │  (1024 → 4096 → 1024)       │      │
  residual│    └──────────────┬──────────────┘      │
     │    │                   │                     │
     └────┤              ⊕ ◄──┘ + x                 │
          │                   │                     │
          └───────────────────┼─────────────────────┘
                              │
                   ┌──────────▼──────────┐
                   │  Final LayerNorm    │
                   │     (1024,)         │
                   └──────────┬──────────┘
                              │
                   ┌──────────▼──────────┐
                   │   Output Linear     │
                   │  (1024, 50257)      │
                   └──────────┬──────────┘
                              │
                   ┌──────────▼──────────┐
                   │      Logits         │
                   │  (seq_len, 50257)   │
                   └──────────┬──────────┘
                              │
                   ┌──────────▼──────────┐
                   │  argmax → Next Token│
                   └─────────────────────┘
```
